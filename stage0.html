I. The Web

  • collection of HTML documents.
  • browser uses HTTP to make requests from servers.

II. HTML (hypertext markup language)

HTML is made up of:

  • text content (what you see)
  • markup (what it looks like)
  • references to other documents
  • links to other pages

A. HTML Markup

i. Tags

  • The basic structure of tags is: <name>contents</name>
  • if you forget to add closing tag, open tag will apply to everything that follows it.
  • scratchpad.io and other editors will autocreate closing tags.

A few useful examples

  • bold tag: <b>contents</b>
  • emphasis tag (italics): <em>contents</em>
  • break tag (inline): <br> (void - does not require closing tag)
  • paragraph tag (block): <p>content</p>

ii. Attributes

The basic structure of attributes is: <tag attr="value"> contents </tag>

A few useful examples

  • Anchor <a> for creating links: <a href="www.reddit.com">derp</a>
  • Image tags (inline): <img src="url" alt="text">

B. Basic HTML Document Structure

<!DOCTYPE HTML>
<html>
<head>
<title>Title!</title>
</head>
<body>
<b>content</b>
</body>
</html>
